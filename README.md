# Fireworks-Trajectory

物理基礎で花火の軌跡を求める問題を Python で解く

## 問題

花火が破裂した場所を原点$ O $として、その破片が$ xy $平面上に$O$を中心として周囲$ 12 $方向に飛び散った。
各方向の初速度を
$ v_0 = 10m/s $
、重力加速度を
$ g\fallingdotseq 10m/s^2 $
として、
$ t = 1s, 2s, 3s, 4s $
後の破片の位置$ (x, y) $を求め、それをグラフに表せ。
ただし、
$ \sqrt{3} \fallingdotseq 1.7 $
とする。
